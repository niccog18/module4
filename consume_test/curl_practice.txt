3. POST a new post
==================================
Command:
curl.exe -X POST https://jsonplaceholder.typicode.com/posts `
  -H "Content-Type: application/json" `
  -d "{\"title\":\"My Test Post\",\"body\":\"This is my custom post body\",\"userId\":1}" `
| python -m json.tool

or 

curl.exe -X POST https://jsonplaceholder.typicode.com/posts -H "Content-Type: application/json" -d "{\"title\":\"My Test Post\",\"body\":\"This is my custom post body\",\"userId\":1}" | python -m json.tool

Output:
[
    {
        "title": "My First API Post",
        "body": "This is my practice post using curl.",
        "userId": 1,
        "id": 101
    }
]
